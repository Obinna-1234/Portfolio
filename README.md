## 🚀 How to Use:  

1. **Clone the Repository**:  
   ```bash
   git clone https://github.com/Obinna-1234/Portfolio.git
   ```

2. **Navigate to the Folder**:  
   ```bash
   cd Portfolio
   ```

3. **Run the Website**:  
   Open the `index.html` file in your browser or deploy it to your favorite hosting platform.
